# ai.hako.tokyo

Coding Agent（Claude Code / GitHub Spec Kit / AWS AI-DLC など）を活用したソフトウェア開発に関するナレッジベース。

外部の重要な情報源をローカルに取り込み、整理・要約したノートと並べて管理する。毎回ネットを取りに行かずに済むようにすることが目的。

## ディレクトリ

- `sources/` — 一次情報の原文・要約（ベンダー別）
- `frameworks/` — ベンダーごとの開発手法の整理
- `topics/` — ベンダー横断のトピック別整理
- `playbooks/` — 実践用の作業手順・レシピ

詳しい運用ルール・frontmatter 規約は [CLAUDE.md](./CLAUDE.md) を参照。
