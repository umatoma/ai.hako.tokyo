# sources/dify/

Dify（LLM アプリケーション開発プラットフォーム）に関する一次情報を集める。

## カテゴリ

- `blog/` — Dify 公式ブログ・リリースノート
- `docs/` — 公式ドキュメント・チュートリアル

## 想定する範囲

- ワークフロー・エージェント機能の設計指針
- プロンプトオーケストレーションのパターン
- 連携可能なデータソース・ツール
- セルフホスト構成・運用上の留意点
