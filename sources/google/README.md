# sources/google/

Google の AI プロダクト・モデル群に関する一次情報を集める。

## カテゴリ

- `blog/` — Google AI / DeepMind / Google Cloud の公式ブログ記事
- `docs/` — 公式ドキュメント・ホワイトペーパー（PDF は Markdown 変換）

## 想定する対象プロダクト

- **Gemini** — Google の汎用 LLM。API・Gemini アプリ。
- **NotebookLM** — ドキュメントを基にした AI ノート / リサーチアシスタント。
- **NanoBanana** — 画像生成・編集モデル（Gemini 2.5 Flash Image の通称）。
