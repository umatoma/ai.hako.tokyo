# sources/n8n/

n8n（オープンソースのワークフロー自動化ツール）に関する一次情報を集める。

## カテゴリ

- `blog/` — n8n 公式ブログ・リリースノート
- `docs/` — 公式ドキュメント・チュートリアル
- `templates/` — 公式テンプレート集の参照（必要なら）

## 想定する範囲

- AI ノード（LLM, ベクトル DB, エージェント）の使い方
- 業務領域別のワークフロー設計パターン（営業・マーケ・人事連携など）
- セルフホスト構成・実行スケジューリング
- 他ツール（Slack / Google Workspace / DB / CRM）との連携

取り込み手順は [`../../playbooks/knowledge-base/01-外部情報を取り込む.md`](../../playbooks/knowledge-base/01-外部情報を取り込む.md) を参照（パスは構造変更後）。
